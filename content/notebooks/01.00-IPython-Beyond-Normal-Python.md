# Jupyter: Beyond Normal Python


有许多Python开发环境可供选择，我经常被问到我在自己的工作中使用哪一个。

我的回答有时会让人感到惊讶：我偏爱的环境是[IPython](http://ipython.org/)加上一个文本编辑器（根据我的心情，可能是Emacs或VSCode）。

Jupyter起初是作为IPython shell而创建的，该shell由Fernando Perez于2001年开发，是一种增强版的Python解释器，此后发展成为一个旨在提供“研究计算整个生命周期工具”的项目，正如Perez所言。

如果说Python是我们数据科学任务的引擎，那么你可以把Jupyter视为互动控制面板。

除了作为Python的实用交互界面外，Jupyter还为语言提供了一些有用的语法扩展；我们将在这里介绍其中最有用的一些扩展。

也许Jupyter项目提供的最熟悉接口就是Jupyter Notebook，这是一种基于浏览器的环境，非常适合用于开发、协作、共享甚至发布数据科学结果。

作为Notebook格式实用性的例子，不妨看看你正在阅读的页面：本书的全部手稿都是以一组Jupyter notebooks形式编写而成。

本书这一部分将通过一些对数据科学实践有帮助的Jupyter和IPython功能进行逐步讲解，特别关注它们所提供超出标准Python特性的语法。

接下来，我们将深入探探讨一些更为实用的*魔法命令*，这些命令能够加速创建和使用数据科学代码中的常见任务。

最后，我们将简要讨论Notebook的一些功能，使其在理解数据和分享结果方面变得更加有效。
