# 输入和输出历史：




之前你看到，IPython shell 允许你通过上箭头和下箭头键访问以前的命令，或者使用 Ctrl-p/Ctrl-n 快捷键。

此外，在 shell 和笔记本中，IPython 提供了多种方法来获取以前命令的输出，以及这些命令的字符串版本。

我们将在这里进行探讨。


## IPython的输入和输出对象

到现在为止，我想你已经熟悉了IPython使用的`In [1]:`/`Out[1]:`风格提示。

但事实证明，这些不仅仅是美观的装饰：它们提供了如何在当前会话中访问之前输入和输出的线索。

假设我们开始一个看起来像这样的会话：

```ipython
In [1]: import math

In [2]: math.sin(2)
Out[2]: 0.9092974268256817

In [3]: math.cos(2)
Out[3]: -0.4161468365471424
```


我们导入了内置的 `math` 包，然后计算了数字 2 的正弦和余弦。

这些输入和输出在终端中以 `In`/`Out` 标签显示，但不仅如此——IPython 实际上创建了一些名为 `In` 和 `Out` 的 Python 变量，这些变量会自动更新以反映这一历史记录。

```ipython
In [4]: In
Out[4]: ['', 'import math', 'math.sin(2)', 'math.cos(2)', 'In']

In [5]: Out
Out[5]:
{2: 0.9092974268256817,
 3: -0.4161468365471424,
 4: ['', 'import math', 'math.sin(2)', 'math.cos(2)', 'In', 'Out']}
```


`In` 对象是一个列表，用于按顺序跟踪命令（列表中的第一个项目是占位符，以便 `In [1]` 可以引用第一个命令）：

```ipython
In [6]: print(In[1])

import math
```

`Out` 对象不是列表，而是将输入数字映射到其输出的字典（如果有的话）：

```ipython
In [7]: print(Out[2])

0.9092974268256817
```

请注意，并非所有操作都有输出：例如，`import` 语句和 `print` 语句不会影响输出。

后者可能令人惊讶，但如果考虑到 `print` 是返回 `None` 的函数，这就说得通；为了简洁起见，任何返回 `None` 的命令都不会被添加到 `Out`.

这在您想与过去的结果进行交互时非常有用。

例如，让我们使用之前计算的结果检查 `sin(2) ** 2` 和 `cos(2) ** 2` 的和：

```ipython
In [8]: Out[2] ** 2 + Out[3] ** 2

Out[8]: 1.0
```

结果为 `1.0`, 正如我们从著名的三角恒等式中所期望的一样。

在这种情况下，使用这些先前的结果可能并不必要，但如果如果您执行了非常耗费资源的计算而忘记将结果分配给变量，它会变得相当方便。


## Underscore Shortcuts and Previous Outputs

The standard Python shell contains just one simple shortcut for accessing previous output: the variable `_` (i.e., a single underscore) is kept updated with the previous output. This works in IPython as well:

```ipython
In [9]: print(_)
1.0
```

But IPython takes this a bit further—you can use a double underscore to access the second-to-last output, and a triple underscore to access the third-to-last output (skipping any commands with no output):

```ipython
In [10]: print(__)
-0.4161468365471424

In [11]: print(___)
0.9092974268256817
```

IPython stops there: more than three underscores starts to get a bit hard to count, and at that point it's easier to refer to the output by line number.

There is one more shortcut I should mention, however—a shorthand for `Out[X]` is `_X` (i.e., a single underscore followed by the line number):

```ipython
In [12]: Out[2]
Out[12]: 0.9092974268256817

In [13]: _2
Out[13]: 0.9092974268256817
```


## 抑制输出

有时您可能希望抑制语句的输出（这在我们将在[Matplotlib简介](04.00-Introduction-To-Matplotlib.ipynb)中探讨的绘图命令中最为常见）。

或者，您执行的命令产生了一个结果，而您不希望将其存储在输出历史记录中，这样可以在其他引用被移除时进行释放。

抑制命令输出的最简单方法是在行末添加分号：

```ipython
In [14]: math.sin(2) + math.cos(2);
```

结果会静默计算，且不会显示在屏幕上，也不会存储在`Out`字典中：

```ipython
In [15]: 14 in Out

Out[15]: False
```


## 相关魔法命令

要一次性访问一批之前的输入，`%history` 魔法命令非常有用。

以下是打印前四个输入的方法：

```ipython
In [16]: %history -n 1-3

1: import math

2: math.sin(2)

3: math.cos(2)
```

如往常一样，您可以输入 `%history?` 获取更多信息以及可用选项的描述（有关 `?` 功能的详细信息，请参见 [IPython中的帮助和文档](01.01-Help-And-Documentation.ipynb)）。

其他有用的魔法命令包括 `%rerun`，它将重新执行部分命令历史，以及 `%save`，它将某些命令历史保存到文件中。
